# PixelPortra — Retro Film Look for Google Pixel

PixelPortra is a custom **Snapseed** photo filter recipe designed specifically to balance the clinical sharpness and high-contrast digital HDR look typical of **Google Pixel** cameras. 

It shifts cool Pixel blues to warm, organic amber tones, introduces a soft cinematic vignette, and overlays natural 35mm-style film grain to replicate classic analog film chemistry.

This repository hosts the promotional GitHub Pages website for the recipe, featuring:
* **Interactive Slider**: A before/after comparison slider showing PixelPortra in action.
* **Sample Gallery**: A clean, responsive grid of Google Pixel photos showing real-world results.
* **Lightbox View**: Instant full-size photo viewing with smooth overlay animations.
* **Recipe QR Code**: A custom QR code that users can scan inside the free Snapseed mobile app to immediately import and apply the recipe to their own photos.

---

## 🎨 Design & Style
The website features a premium design inspired by the classic **Fujifilm PRO Neg. Hi** film canister:
* **Color Palette**: Dark charcoal backgrounds (`#0C0C0C`), elegant warm brass/gold accents (`#C5A87F`), and simulation accents using vibrant green and deep violet.
* **Typography**: Plus Jakarta Sans (sans-serif) and Playfair Display (serif).
* **Authentic Texture**: A subtle, hardware-accelerated static film grain overlay.

---

## 🛠️ Usage & Deployment

1. **Local Editing**: You can make updates directly in `index.html`. It uses the Tailwind CSS CDN for easy styling.
2. **Adding Gallery Images**: Add your photos under `media/` or `media/v2/` and reference them in the grid section within `index.html`.
3. **Deployment**: Since the repository is configured for GitHub Pages, any commits pushed to the `main` branch are instantly deployed.
